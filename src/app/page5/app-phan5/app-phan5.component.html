Truy vấn dữ liệu

Truy vấn dữ liệu là một trong các thao tác cơ bản, được thực hiện bởi câu lệnh SELECT. Đây là câu lệnh có cú pháp phức tạp. Những mệnh đề chính của SELECT là :

    SELECT

Retrieves rows from the database and allows the selectic of one or many rows or columns from one or many tables . The full Syntax of the SELECT statement is complex , but the main clauses can be summarized as :

    SELECT select list [ INTO new_table ]

    FROM table_source_list

    [WHERE search _ condition ]

    [ GROUP BY group_by_expression ] [ HAVING search _ condition ]

    [ ORDER BY order_expression [ ASC | DESC ]
