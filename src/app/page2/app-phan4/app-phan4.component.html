PHÉP TOÁN QUAN HỆ

4.1 Phép chọn (selection)

Xét R là quan hệ trên lược đó quan hệ. Ký hiệu E là một biểu thức logic gồm các phép toán số học và logic <, <=, >, >=, ≠, ∧, v, ¬ đặt tả một điều kiện.

Phép chọn σ trên quan hệ R theo điều kiện E, ký hiệu σE ( R ) có kết quả là một tập hợp con của R, gồm các bộ t ∈ R thỏa điều kiện E:

          σE ( R ) = {t/ t ∈ R ∧ t(E) }, ký hiệu t(E) nghĩa t thỏa điều kiện E

4.2. Phép chiếu ( Projection )

Xét R là quan hệ trên lược đó quan hệ . X là một tập hợp con khác rỗng

Phép chiếu quan hệ R trên X , ký hiệu π X (R ) có kết quả là tập hợp các bộ t ∈ R nhưng chỉ lấy giá trị trên X .

Phép chọn σ trên quan hệ R theo điều kiện E, ký hiệu σE ( R ) có kết quả là một tập hợp con của R, gồm các bộ t ∈ R thỏa điều kiện E:

          π X (R ) = { t/ t ∈ R ∧ t[E ] } , ký hiệu t[X] nghĩa t nhận giá trị trên X

Chú ý rằng các bộ giống nhau trong bảng kết quả chi được giữ lại một.
