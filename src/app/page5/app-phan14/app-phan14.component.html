Tạo dự phòng và phục hồi cơ sở dữ liệu

14.1. Sao lưu một cơ sở dữ liệu đã có ( Backup )

BACKUP DATABASE { DatabaseName } TO DISK = {BackupFileName }

  DatabaseName

    Đặc tả cơ sở dữ liệu đã có

   BackupFileName

    Đặc tả đường dẫn đến nơi lưu trữ file dự phòng

Ví dụ: Cơ sở dữ liệu qlnv đã có , tạo file dự phong qlnv _ backup.bak trên ổ đĩa G :

  BACKUP DATABASE qlnv TO DISK = ' G:\glnv backup.bak '
