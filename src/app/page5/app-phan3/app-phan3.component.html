Tạo và cập nhật cơ sở dữ liệu

3.1 Để tạo một cơ sở dữ liệu người ta dùng câu lệnh
CREATE DATABASE

Creates a new database and the files used to store the database , or attaches a database from the files of a previously created database .

Syntax

CREATE DATABASE ( database _ name )

Ví dụ , tạo cơ sở dữ liệu tên qlnv CREATE DATABASE qlnv

Lệnh này tạo ra hai file hệ thống là : qlnv . mdf và Qlnv_log . ldf
